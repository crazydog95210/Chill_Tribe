Resido Real Estate Template Documentation

1. Introduction

Resido is a premium HTML5 template for real estate related websites which is built with the lates bootstrap version. This template is suitable for any agency, agent, property, real estate, rental, retina ready, real estate, corporate, apartment. The whole template is made with excellent responsiveness.


We put a lot of love and effort to make Resido a useful template for everyone and now It comes with 8+ unique home page & colors. We are keen to release continuous long term updates and lots of new features will be coming soon in the future releases. Once you purchased Resido Template, you will be entitled to free download of all future updates for the same license.
